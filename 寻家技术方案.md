## 寻家（Find Home）技术方案

### 核心架构

```
┌──────────────────────────────────────────┐
│  框架：Next.js 16 (App Router)            │
│  部署：Vercel（零配置，Git推送自动部署）     │
│  UI：shadcn/ui + Tailwind CSS v4         │
│  主题：next-themes（亮色/暗色/极简）       │
│  图标：Lucide React                       │
│  包管理：pnpm                             │
│  待接入：Supabase（数据库+认证）           │
│  AI接口：兼容OpenAI格式（可切换供应商）    │
│  模式：买房/租房双模式切换                 │
│  地图：高德地图 JS API 2.0               │
└──────────────────────────────────────────┘
```

## 项目结构

```
findhome/
├── app/
│   ├── api/parse-screenshot/  # AI 截图识别 API Route
│   ├── layout.tsx              # 根布局（主题、字体、Analytics）
│   ├── page.tsx                # 主页面（状态管理、组件组装）
│   └── globals.css             # 全局样式 + CSS变量
├── components/
│   ├── find-home/
│   │   ├── login-page.tsx     # 登录/注册页
│   │   ├── header.tsx          # 顶部导航（搜索、模式切换）
│   │   ├── property-table.tsx  # 房源大表格（核心组件）
│   │   ├── property-detail.tsx # 房源详情面板
│   │   ├── property-compare.tsx# 对比模式面板
│   │   ├── property-map.tsx   # 地图视图（高德地图标注+信息卡片）
│   │   └── floating-action-button.tsx # 悬浮按钮（手动添加/截图识别）
│   ├── auth-provider.tsx      # 认证Provider
│   ├── theme-provider.tsx      # 主题Provider
│   └── ui/                     # shadcn/ui 组件库
├── hooks/
│   └── use-properties.ts      # 房源数据 Hook（CRUD + Supabase）
├── lib/
│   ├── ai.ts                  # AI 截图识别（前端调用）
│   ├── amap.ts                # 高德地图加载 + POI 搜索
│   ├── supabase.ts            # Supabase 客户端
│   ├── storage.ts             # 图片上传/删除（Supabase Storage）
│   ├── db-transforms.ts       # 数据库字段映射（camelCase <-> snake_case）
│   └── utils.ts                # 工具函数
├── types/
│   └── property.ts             # 类型定义（Property、ViewMode、ColumnConfig等）
└── supabase/
    └── schema.sql              # 数据库建表 SQL + 触发器 + RLS 策略
```

## 开发路线图

### 已完成：前端MVP + 数据持久化 + AI 截图识别

- 房源大表格（列表/编辑/对比三种模式，表头滚动固定）
- 列设置拖拽排序、列宽拖拽调整、标签列自适应宽度换行
- 房源详情面板（基础信息 + 看房记录时间轴 + AI分析占位）
- 排序、搜索、标签筛选、列级筛选
- 自定义列、收藏、状态管理
- Supabase 数据库 + 用户认证（用户名注册/登录）
- RLS 行级安全策略，数据隔离
- 注册自动生成示例数据，支持一键清除
- 图片上传（Supabase Storage）
- 截图识别自动录入房源（AI 多模态模型，兼容 OpenAI 格式接口），识别数据一次性入库无闪烁，截图自动设为封面图
- 房号字段（栋数+房间号），默认显示在小区名旁，同一小区多套房清晰区分
- 封面图从顶部截取展示，点击可全屏查看；非必填字段为空时显示"-"
- 移动端响应式适配（详情/对比全屏弹窗、搜索栏折叠、主题切换移入用户菜单）
- 地图定位（高德地图 JS API，POI 搜索自动定位，城市选择持久化到数据库，坐标缓存，同小区重叠处理，QPS 限流）
- 落地页优化（场景化标题、差异化痛点、核心亮点分层、信任要素）
- 买房/租房双模式（Header 切换按钮，模式持久化，价格单位自动适配，AI prompt 按模式调整，房源按 mode 分开管理）

### 下一步：AI 增强 + 体验优化

- Vercel AI SDK 接入 OpenAI，替换 Mock AI 分析数据
- 多房源对比智能报告
- 导出 Excel/PDF

```bash
pnpm build
```
